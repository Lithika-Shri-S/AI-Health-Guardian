# AI-Health-Guardian
AI Health Guardian is a smart health assistant that helps users check symptoms, detect emergencies, scan medicines, and interact through a voice assistant, making healthcare guidance quick, simple, and accessible.

Tech Stack

Frontend:
	•	React.js / HTML, CSS, JavaScript

Backend:
	•	Node.js / Flask (Python)

AI/ML:
	•	Python
	•	TensorFlow / Scikit-learn

Database:
	•	Firebase / MongoDB

APIs & Tools:
	•	Wearable APIs (Google Fit / smartwatch data)
	•	Twilio (for alerts)
	•	OpenAI API (for chatbot)

 “AI Health Guardian”

Features:
	•	Symptom checker (AI)
	•	Emergency severity detection
	•	Medicine scanner
	•	Voice assistant
